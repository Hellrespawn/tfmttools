# Report Viewer Assets

The report viewer vendors the standalone HTM/Preact browser module from
esm.sh so generated reports work without network access.

Pinned source:

- `https://esm.sh/htm/preact/standalone`
- Resolved module:
  `https://esm.sh/htm@3.1.1/es2022/preact/standalone.mjs`
- HTM version: `3.1.1`
- Preact version bundled by esm.sh at refresh time: `10.27.2`

Refresh command:

```sh
scripts/vendor-report-assets.sh
```
